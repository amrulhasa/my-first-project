# BDTechMarket

## Project Description

BDTechMarket is an ASP.NET Core MVC web application for managing a tech marketplace.
Users can browse products, view details, and manage technology-related items.

## Technologies Used

* ASP.NET Core MVC
* C#
* HTML
* CSS
* Bootstrap

## Project Structure

* Controllers → Handle application logic
* Models → Data models
* Views → User interface
* wwwroot → Static files (CSS, JS, images)

## How to Run

1. Clone the repository
2. Open the project in Visual Studio
3. Run the application
